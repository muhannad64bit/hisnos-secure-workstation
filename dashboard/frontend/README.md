# HisnOS Dashboard UI Skeleton

This folder contains early UI scaffolding for the HisnOS governance dashboard.

Current state model endpoint:
- `GET http://127.0.0.1:7374/api/system/state`

What exists now:
- `system-state.html`: simple client-side page (no framework) to validate the control-plane state model end-to-end.

Next (planned):
- Migrate to SvelteKit routes once the frontend toolchain is initialized.
